README

An diesem Spiel wird noch gearbeitet, daher ist manches unvollständig und es können möglicherweise Fehler auftreten. 

Dieses Spiel ist auf Deutsch und Englisch verfügbar. Man kann zwischen beidem im Optionsmenü wechseln. 
(Es gibt einige unvollständige Übersetzungen, daher wird mancher Text in Englisch angezeigt auch wenn das Spiel auf Deutsch gestellt ist.)

Level 1 bis 25 können freigeschalten werden und sind spielbar.

Nur in dieser Demo verfügbar, um meine Arbeit zu zeigen: Im Optionsmenü gibt es ein Eingabefeld, in das man eine Nummer eingeben kann und die Levels bis dahin freischalten kann. 
Wenn die Zahl über 25 ist, bleiben die Levels trotzdem gesperrt, aber neue Informationen in“ How to Play“ wird freigeschallten.



This Game is still being worked on so there is some incomplete content and the possibility of some bugs.

This game is available in both German and English. You can switch between these in the options menu. 
(There are some incomplete translations so some text will appear in English even if the game is set to German.)

Levels 1 to 25 can be unlocked and are playable.

Just in this Demo, so you can see some more of my work: Located in the options menu is an input field where you can input a number to unlock up to that level. 
If the number is over 25 those levels will remain locked however information in the How to Play menu will become available.
